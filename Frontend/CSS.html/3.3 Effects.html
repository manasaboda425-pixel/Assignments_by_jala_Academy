Static Position

An element with position: static; flows normally with the rest of the page layout. It does not take top/bottom/left/right properties:

This box uses position: static;

Relative Position

An element with position: relative; is placed relative to where it would normally be:

This box uses position: relative;

Fixed Position

An element with position: fixed; stays firmly in one spot on the screen, even if you scroll down the page:

Fixed Item

An element with position: absolute; places itself relative to the nearest positioned parent container:
